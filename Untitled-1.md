PALETTE DE COMMANDES:
┌───────────────────────┬─────────────────────────────────────┐
│ Raccourci │ Action │
├───────────────────────┼─────────────────────────────────────┤
│ Ctrl + Shift + P │ Ouvrir palette de commandes │
│ F1 │ Ouvrir palette de commandes │
└───────────────────────┴─────────────────────────────────────┘

La palette de commandes (Ctrl + Shift + P) donne accès à TOUTES les
fonctionnalités de VSCode. Tapez simplement ce que vous cherchez!

Exemples:

- "Format Document"
- "Change Language Mode"
- "Transform to Uppercase"
- "Reload Window"
